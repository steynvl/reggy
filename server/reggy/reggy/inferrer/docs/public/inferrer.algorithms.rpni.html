class inferrer.algorithms.rpni.rpni.RPNI(pos_examples: Set[str], neg_examples: Set[str], alphabet: Set[str])[source]¶

Bases: inferrer.algorithms.algorithm.Algorithm

An implementation of the Regular Positive and Negative Inference (RPNI) algorithm. This algorithm tries to make sure that some generalisation takes place and, in the best case, returns the correct target automaton.

Learns the grammar from the sets of positive and negative example strings. This method returns a DFA that is consistent with the sample.
